```
usage: closetrade [-i ORDERID] [-u UNITS] [-h]
```

Close a trade by id.

```
optional arguments:
  -i ORDERID, --id ORDERID
			The Trade ID to close.
  -u UNITS, --units UNITS
			The number of units on the trade to close. If not set it defaults to all units.
  -h, --help            Show this help message
```
