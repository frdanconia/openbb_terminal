```
usage: forecast [-j] [-n] [-d] [-h]
```

Shows forecast dashboard

```
optional arguments:
  -j, --jupyter   Shows dashboard in jupyter-lab. (default: False)
  -n, --no-input  Skips confirmation to run server. (default: False)
  -d, --dark      Whether to show voila in dark mode (default: False)
  -h, --help      show this help message (default: False)
```
